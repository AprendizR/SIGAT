# 📦 SIGAT - Sistema de Gestão de Atividades

Projeto desenvolvido como parte da formação no curso de **Gestão da Tecnologia da Informação** da **Fatec Itu**. O objetivo deste sistema é auxiliar na organização, controle e gestão de atividades, permitindo uma visualização clara do fluxo de trabalho e melhoria na produtividade.

---

## 🚀 Tecnologias Utilizadas

* **HTML5** (Estruturação)
* **CSS3** (Estilização e Responsividade)
* **JavaScript** (Lógica de aplicação e manipulação do DOM)
* **LocalStorage** (Persistência de dados local)

---

## 📋 Funcionalidades Principais

* **Cadastro de Atividades:** Permite a criação de novas tarefas com descrição e status.
* **Gerenciamento:** Edição e exclusão de itens conforme a necessidade.
* **Persistência de Dados:** O sistema salva suas atividades automaticamente no navegador, garantindo que nada seja perdido ao recarregar a página.
* **Interface Responsiva:** Desenvolvido pensando na usabilidade em diferentes tamanhos de tela.
